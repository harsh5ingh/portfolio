# Harsh Singh – Portfolio

A modern, responsive personal portfolio website built with vanilla HTML, CSS, and JavaScript.

## ✨ Features

- **Dark theme** with purple accent and smooth animations
- **Responsive** – works on mobile, tablet, and desktop
- **Sections**: Hero, About, Skills, Projects, Contact
- **Typing animation** in the hero section
- **Scroll-reveal animations** for all sections
- **Animated skill bars** triggered on scroll
- **Animated counters** in the About section
- **Mobile navigation** with hamburger menu
- **Contact form** (front-end only demo)

## 🚀 Tech Stack

Built with plain HTML5, CSS3 (custom properties, grid, flexbox), and vanilla JavaScript (IntersectionObserver, CSS animations). No frameworks or build tools required.

## 🗂️ File Structure

```
portfolio/
├── index.html   # Main page
├── style.css    # All styles
├── script.js    # Interactivity & animations
└── README.md
```

## 🌐 Live Demo

Open `index.html` in any modern browser, or deploy to GitHub Pages / Netlify / Vercel.
